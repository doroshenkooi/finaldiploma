Применен Webpack для экспорта и импорта между модулями.
В javaScript применены:

- работа с DOM;

- циклы и итерирование;
- fetch API;
- условные операторы;
- тернарные операторы;
- динамическое обновление интерфейса;
- обработчики событий;
- управление вложенными элементами;
